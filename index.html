Installation

Connecting with database

Find config.ini in class directory, set hostname, dbname, username and password

Example
hostname = localhost;
dbname = joomshaper;
username = root;
dbpassword = password;

Thats it, it will install and create database automatically when you browse home page of this project. No need import any database

Note: dont use config value in double quotation (""), even if there is no password
